# 🧩 Programação Orientada a Objetos

Este módulo contém exercícios práticos de **Programação Orientada a Objetos (POO)** em Java. Cada exercício explora um cenário diferente para aprofundar os conceitos fundamentais da POO.

## 📚 O que é POO?

Programação Orientada a Objetos é um paradigma de programação baseado no conceito de "objetos", que combinam dados (atributos) e comportamentos (métodos). Os pilares fundamentais são:

- **Encapsulamento**: ocultação dos detalhes internos de uma classe
- **Herança**: reutilização e extensão de comportamentos entre classes
- **Polimorfismo**: capacidade de tratar objetos de tipos diferentes de forma uniforme
- **Abstração**: modelagem de conceitos do mundo real como classes e interfaces
