# Biblioteca Raro Academy

## Descrição do Projeto
Este projeto é uma aplicação de gerenciamento de biblioteca desenvolvida como parte do treinamento de Java da Raro Academy. O objetivo é fornecer uma solução que permita aos usuários gerenciar empréstimos, devoluções e o catálogo de livros disponíveis.

## Funcionalidades
- Gestão de livros (adição, remoção, edição)
- Registro de empréstimos e devoluções
- Busca de livros por título e autor
- Notificações de atraso de devolução

## Tecnologias Utilizadas
- Java
- Spring Boot
- Banco de Dados MySQL

## Instruções de Execução
Para rodar a aplicação, clone o repositório e execute o comando:
```
./mvnw spring-boot:run
```
