# Sistemas de Persistência de Dados

Os sistemas de persistência de dados são fundamentais para o armazenamento e recuperação de informações em aplicações. Esses sistemas garantem que os dados sejam mantidos de forma confiável, mesmo após o término de processos ou desligamento do sistema.

## Tipos de Sistemas de Persistência

1. **Bancos de Dados Relacionais:** Utilizam tabelas para armazenar dados. Exemplos incluem MySQL, PostgreSQL e Oracle.
2. **Bancos de Dados Não Relacionais (NoSQL):** Projetados para armazenar grandes volumes de dados não estruturados. Exemplos incluem MongoDB, Cassandra e Redis.
3. **Sistemas de Arquivos:** Armazenam dados em arquivos localmente ou em servidores. Utilizados para aplicações que não necessitam de funcionalidades avançadas de gerenciamento de dados.

## Importância

A escolha de um sistema de persistência de dados deve considerar fatores como escalabilidade, consistência, durabilidade e desempenho, visando atender às necessidades específicas da aplicação em desenvolvimento.
